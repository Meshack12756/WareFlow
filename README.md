# 🏢 WareFlow – Enterprise Warehouse Management System

**WareFlow** is a full‑stack warehouse management solution that integrates sales processing, inventory control, employee performance tracking, and real‑time analytics – all in one clean, modern platform.

Built for logistics teams and warehouse operators, WareFlow helps you track stock levels, process sales, manage employee performance, and gain actionable insights with minimal friction.

## ✨ Key Features

- **Product Management** – Full CRUD with search, filtering, and low‑stock alerts.
- **Sales Processing** – Create sales with atomic inventory deduction and automatic profit calculation.
- **Employee Management** – Manage staff, assign pay models (commission, hourly + bonus, profit‑share), and track timecards.
- **Performance Dashboards** – Real‑time KPIs, sales summaries, and employee performance charts.
- **Refund Handling** – Process full/partial refunds with optional restocking.
- **Search & Filters** – Quickly find products, sales, or employees by multiple criteria.
- **Reports & Analytics** – Top‑selling products, low‑stock items, daily sales trends, and employee performance.
- **Redis Caching** – High‑performance caching for product listings and inventory checks.
- **Modern UI** – Built with Material‑UI, featuring smooth animations and responsive design.
